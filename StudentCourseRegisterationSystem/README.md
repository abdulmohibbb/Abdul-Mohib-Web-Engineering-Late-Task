# Abdul-Mohib-Web-Engineering-Late-Task

Student Course Registration System built with Django and SQLite.

## Run

1. Open the `StudentCourseRegisterationSystem` folder.
2. Activate the virtual environment in `venv`.
3. Run `python manage.py migrate`.
4. Run `python manage.py runserver`.
